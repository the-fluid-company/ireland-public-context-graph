# Ireland Public Context Graph

An open-source contextual intelligence layer for publicly available Irish datasets.

The project collates, normalizes, versions, and links public data across Irish domains such as transport, roads, planning, environment, weather, demographics, health, education, public services, infrastructure, and local authorities. Road safety is one domain inside the broader context layer.

The project provides data, relationships, provenance, schemas, exports, and hosted MCP access. It does not provide conclusions, causation, legal opinions, engineering assessments, fault claims, official designations, or recommendations.
